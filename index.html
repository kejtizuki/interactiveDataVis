What is the point of scales?
The point of scales is to map the numbers from input domain to output range. Nomralize the numbers,
so map the numeric value to another numeric value that is inside range.

d3.min() and d3.max() are useful because they allow to find values for domains for different datasets.

Margin convention proposed by Mike Bostock syas to first define all marigns, include them in width and barHeight
and than define svg with attribute "transform" that translates it taking into considerations the defined margins.
By doing that margins can be omitted later on.
